24.2 #Text Stripe - UI Text Stripe

Toggle full screen Open in new window Toggle example guides Toggle HTML markup

A stripe is a very versatile full-width container for any content. The contents are padded and depending on breakpoint, provide different fixed container sizes The calculation looks like this:

    @media screen and (min-width: @breakpoint-sm) {
        width: calc(@breakpoint-sm * (min(1, min(@width + .2, 1))) ~'- 1em');
    }
    @media screen and (min-width: @breakpoint-md) {
        width: calc(@breakpoint-md * (min(1, min(@width + .05, 1))) ~'- 1em');
    }
    @media screen and (min-width: @breakpoint-lg) {
        width: calc(@breakpoint-lg * @width ~'- 1em');
    }
