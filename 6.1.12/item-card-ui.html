8.2 #Card - UI Card

Toggle example guides Toggle HTML markup

A card is a box with colorful background and some padding Usually containing a title, description and actions (buttons/icons) Cards contain little and scannable content to quickly get to the desired full content. Thus, a description should contain no more than 140 letters
